Rockaway Ventures

Meeting Prep Skill

A private daily preparation packet for Ventures meetings, built from calendar context and the read-only Rockaway Ventures Brain.

PDF packet + markdown source + compact chat summary

RV

What It Does

  • Looks at the user's connected calendar/account context.
  • Prepares every in-scope Ventures meeting, with `today` meaning remaining meetings from now onward.
  • Retrieves context from the read-only Rockaway Ventures MCP brain.
  • Creates a dense executive PDF packet designed to read quickly before calls.

What It Produces

  • PDF: the primary artifact, with no source trail.
  • Markdown: editable source with full source trail.
  • Chat: a short summary with file links and top priorities.
  • JSON/HTML: structured source and preview files.

Default Behavior

The skill creates one all-day packet by default. If run halfway through the day, it prepares upcoming meetings only. Use "whole day" or "include past meetings" when you want the full date.

It excludes declined and cancelled events, includes tentative events with a status marker, and creates only a stub for private/busy events with no details.

1Install The Skill

One command
npx -y skills@latest add zabrodsk/rockaway-ventures-meeting-prep-skill -g --agent codex claude-code --copy -y --full-depth && "$HOME/.agents/skills/rockaway-ventures-meeting-prep/setup.command"

This installs the skill for Codex and Claude Code, then runs setup.

Message you can send to Codex or Claude
please install the meeting prep skill from the GitHub repo and run the setup command. This is my bearer token:
After setup, use
$rockaway-ventures-meeting-prep prep me for today
$rockaway-ventures-meeting-prep prep me for my next meeting
$rockaway-ventures-meeting-prep prep me for tomorrow

2Connect The Ventures Brain

Ask the Rockaway brain admin for your Ventures bearer token. Setup connects the Ventures brain automatically. The manual commands below are here for reference.

What the setup connects
rockaway-ventures
http://100.102.180.108:8789/rockaway-ventures/mcp
Claude Code manual setup
claude mcp add rockaway-ventures --transport http \
  http://100.102.180.108:8789/rockaway-ventures/mcp \
  --header "Authorization: Bearer USER_TOKEN"
Codex manual setup
export ROCKAWAY_VENTURES_MCP_TOKEN="USER_TOKEN"
codex mcp add rockaway-ventures \
  --url http://100.102.180.108:8789/rockaway-ventures/mcp \
  --bearer-token-env-var ROCKAWAY_VENTURES_MCP_TOKEN
codex mcp get rockaway-ventures
Codex config equivalent
[mcp_servers.rockaway-ventures]
url = "http://100.102.180.108:8789/rockaway-ventures/mcp"
bearer_token_env_var = "ROCKAWAY_VENTURES_MCP_TOKEN"

The environment variable must be available to Codex when it starts.

3Read The Packet

Day Summary

Use the first page to see the main themes, urgent prep items, and the meetings that need the most attention.

Per-Meeting Sections

Each meeting has context, previous discussion, open threads, people notes, sensitive signals, suggested talk track, questions, and watchouts.

Recommendations

Suggested agendas and talk tracks are marked as recommendations, not retrieved facts.

Audit Trail

The PDF omits source details for readability. Open the markdown file when you want the full source trail.

Output Location

~/Rockaway Meeting Briefs/ventures/YYYY-MM-DD/
  rockaway-ventures-meeting-prep-YYYY-MM-DD.pdf
  rockaway-ventures-meeting-prep-YYYY-MM-DD.md
  rockaway-ventures-meeting-prep-YYYY-MM-DD.html
  rockaway-ventures-meeting-prep-YYYY-MM-DD.json

Same-day reruns overwrite the default files so the latest packet stays easy to find.

Use It For

  • Partner meetings.
  • Founder or deal calls.
  • Fund administration follow-ups.
  • Board, IC, and portfolio discussions.

Boundaries

  • Read-only brain access.
  • No calendar writeback.
  • No direct brain edits.
  • No source trail in the PDF or chat summary.
